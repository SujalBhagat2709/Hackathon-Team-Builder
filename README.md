# Hackathon Team Builder

## Overview

A platform that helps hackathon participants find teams based on skills.

---

## Features

- Create hackathon teams
- Add members
- Define required skills
- Match users with teams

---

## Files

- team.py
- matcher.py

---

## Usage

```bash
python team.py
python matcher.py
```

---

## Example

```text
Enter Your Skill: Python

Suggested Teams:

AI Builders
```
